¿Que diferencia hay entre cada plaqueta de arduino y que tipos existen?

Las placas de Arduino son diferentes versiones del mismo sistema para programar y controlar componentes electrónicos. Todas sirven para lo mismo pero cambian en tamaño potencia y cantidad de conexiones. El Arduino UNO es el más común y el más usado para empezar porque es sencillo y equilibrado. El Arduino Nano es más pequeño y se usa cuando hay poco espacio. El Arduino Mega es más grande y potente con más pines para conectar muchos sensores o motores. También hay otras placas como el Arduino Leonardo que puede simular un teclado o mouse y algunas con WiFi o Bluetooth para proyectos inalámbricos. En resumen todas hacen lo mismo pero se eligen según el tamaño del proyecto y la cantidad de cosas que quieras conectar.
